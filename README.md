# 🚗 Car Price Analysis Dashboard

## 📊 Project Overview

This project analyzes 10,000 car listings to uncover pricing trends using Excel.

## 🔍 Key Insights

* Car prices have increased over the years
* Low mileage cars are more expensive
* Automatic cars tend to cost more

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
